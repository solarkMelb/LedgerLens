# LedgerLens
 
A local LLM-powered tool for reading moomoo brokerage account statements and answering questions about portfolio performance, profit and loss, and trade history.
 
Statements are parsed into a local database and queried through a command-line interface. 

The tool runs on a local language model by default and can optionally use Claude, OpenAI, or DeepSeek through an API key.

## About the name
 
A ledger is the principal record of an account's financial transactions, historically the central book in which every entry is kept. 
LedgerLens applies a focused view, or lens, over that record so it can be read and questioned directly.
